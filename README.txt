+++ Beschreibung +++

Dieses Script erweitert deine Nutzerdatenbank des Honeypots Kippo um alle möglichen Benutzer/Passwort-Kombinationen, basierend auf den bisherigen Einträgen und deinen Kippo-Logs.

+++ Installation +++

Öffne die index.php und passe in den Zeilen 16 und 17 die Einstellungen an.
In Zeile 16 gibts du den Installationspfad von kippo an (mit / am Ende) und in der 17 kannst du dem Script eine bestimmte Menge RAM zuweisen (je nach Größe ist hier 1G schon sinnvoll).

+++ Anwendung +++

Einfach die index.php aufrufen. Es wird eine Statistik zurückgegeben.
Außerdem werden Backups von der Datenbank und den Logs angelegt. Bitte stelle sicher, dass das Script Schreibrechte hat.
